# AI Components

Components specific to AI provider/model management and interactions.

## Components

- [AIModelsComponent](./AIModelsComponent.md) — Lists available AI models for the selected provider using dynamic API retrieval with static fallback.

## Components that use AI for processing

- [AITextEvaluate.cs](../../../src/SmartHopper.Components/Text/AITextEvaluate.cs) — Evaluates text with AI.
- [AITextGenerate.cs](../../../src/SmartHopper.Components/Text/AITextGenerate.cs) — Generates text with AI.
- [AITextListGenerate.cs](../../../src/SmartHopper.Components/Text/AITextListGenerate.cs) — Generates a list of items with AI.
- [AIListEvaluate.cs](../../../src/SmartHopper.Components/List/AIListEvaluate.cs) — Evaluates lists with AI.
- [AIListFilter.cs](../../../src/SmartHopper.Components/List/AIListFilter.cs) — Filters lists with AI.
- [AIImgGenerateComponent.cs](../../../src/SmartHopper.Components/Img/AIImgGenerateComponent.cs) — Generates images with AI.
